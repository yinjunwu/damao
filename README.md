# 医启诊 - 会员中心前端原型

基于纯 HTML + CSS + JavaScript 实现的会员中心前端交互原型，包含完整的实名认证流程（个人认证、企业法人人脸识别、企业对公打款）和开票系统。

## 功能特性

- **会员信息展示** — 认证状态管理（未认证 / 已实名 / 审核中）
- **购买记录管理** — 订单列表、复选框选择（全选/半选）、实时金额统计
- **开票系统** — 基于订单选择的发票申请，含实名验证和最低金额限制
- **个人实名认证** — 信息填写 → 支付宝扫码 → 异步回调结果判定
- **企业法人人脸识别认证** — 四要素填写 → 法人扫脸 → 回调判定
- **企业对公打款认证** — 企业信息填写 → 打款确认 → 金额验证 → 三态审核结果
- **调试面板** — 支持切换会员类型、强制指定回调结果

## 快速开始

```bash
# 方式一：直接打开
open index.html

# 方式二：使用 HTTP 服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 技术栈

- 纯前端，零依赖
- 单 HTML 文件（内联 CSS + JS）
- SVG 图标 + 模拟异步回调接口
